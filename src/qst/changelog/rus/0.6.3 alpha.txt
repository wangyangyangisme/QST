0.6.2 to 0.6.3 (alpha) Changes

Уведомление.
Версия QST 0.6.x пока еще не является стабильной и готовой к использованию. Она будет таковой, предположительно, через 4-6 минорных релизов.

|====================| GENERAL |===================|
- Исправлено огромное количество ошибок.
- Добавлено несколько новых тестов и обновлены старые.
- Восстановлена работоспособность динамических фильтров.
- Реорганизованы пространства имен.
- Удален механизм автоматических функторов, зависящих от типа значения.
- Проведен масштабный рефакторинг выполнения SELECT-запросов, настройки и моделизации представлений.
- Из подсистемы MVC в подсистему Special перенесены классы ComboBoxSettings, ViewAppearance, ColumnSettings.
- Два перечисления: ViewType и SettingType сделаны глобальными, пересмотрены.
- Несколько переименований в интерфейсах классов ModelViewDescriptor и других.
- Переписана генерация секции WHERE.
- Переписан QueryFrame. При генерации запросов приведены в порядок пробелы и скобки.

|=====================| CODE |=====================|
- Класс QueryAccessory переименован в QueryClauseAccessory.
- Исправлена ошибка в QueryWhere, из-за которой появлялось ключевое слово WHERE при пустой секции.
- Свойство modelColumn перенесено из QstField в ViewAppearance.
- Валидность QueryWhere теперь вычисляется иначе. Исправлен метод QueryWhere::isValid().
- Исправлены недочеты, связанные с explicit-конструктором QueryValue.
- Удалены дублирующие классы QstDefaultItemnamEextractor, QstAbstractItemNameExtractor.
- Namespace QstSpecial переименован в Special.
- Рефакторинг функций QstAbstractModelHandler:::_fieldData().
- Рефакторинг конструкторов QueryField.
- Добавлен класс-обертка FramedQuery.
- Поле token убрано из класса QueryFrame.
- Добавлена функция QstAbstractModelHandler::columnValuesList().
- Другие изменения.

|====================| TESTS |=====================|
- Обновлены тесты для новых случаев DefaultItemNameExtractor.
- Добавлены тесты:
* ut_QueryBatch
* ut_QueryGeneration
* ut_QueryWhere

|====================| FIXES |=====================|
- Исправлена ошибка генерации EXEC-запроса (SELECT function()) с пустыми параметрами.
- Исправлена важная ошибка при обновлении placeholder'ов.
- Исправлена ошибка с получением значения, возвращенного запросом.
- Исправлена ошибка с QueryWhere, которая не позволяла делать более 2 вызовов QueryBatch::where().
- Исправлена ошибка с modelColumn для comboBox.
- Исправлена критическая ошибка в QstAbstractModelHandler, приводящая к падению программы при вызове функции comboBoxFieldValue().
- Исправлены другие многочисленные ошибки.
